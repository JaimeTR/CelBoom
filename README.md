# CelBoom
Herramienta que sirve para mandar un ataque de spam SMS a un número de celular, funciona con impulse.

Instalación:
- git clone https://github.com/Monkey-hk4/CelBoom
- cd CelBoom
- chmod +x CelBoom.sh
- chmod +x install.sh
- bash install.sh
- Para ejecutar  >>> bash CelBoom.sh <<<
